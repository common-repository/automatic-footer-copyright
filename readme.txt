Add automatic copyright information in footer of Genesis or any Genesis child theme.

== Description ==

This plugin adds automatic copyright information in footer of Genesis. It also removes link to Genesis and WordPress. The format is &copy; first-post-year - current-year your-blog-name.

If year of first post is same as current year, it will display only current year. 

The plugin works only with Genesis theme for now. I will add support for other theme frameworks later. If you want have any questions or feature requests, please contact us via http://wpblogexperts.com/contact-us

== Installation ==

To install the plugin, download the zip file to your computer, unzip it and upload the AutoFooterCopyright folder to /wp-content/plugins folder

== Frequently Asked Questions ==

= Which themes does the plugin support? =

Genesis and Genesis child themes for now. 

= How can I customize the text? =

Text can not be changed right now. The feature will be added in future versions.
